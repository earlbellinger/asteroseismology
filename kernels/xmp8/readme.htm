NUMERICAL  METHODS  FOR  SCIENTISTS  AND ENGINEERS

FORTRAN PROGRAMS


This directory contains Fortran 77 programs using various subprograms described in Appendix B. All programs use double precision (REAL*8 / COMPLEX*16) arithmetic. Single and quadruple precision versions of these programs can be found in directories f77/xmp4 and f77/xmp16 respectively. The subprograms can be found in the directory f77/sub8

All these programs have been tested to some extent, but there may be some bugs. Users are welcome to use these programs at their own risk. Any bugs discovered in these routines may be notified to

antia@tifr.res.in

The main purpose of the programs in this directory is to provide concrete examples of using the subprograms described in Appendix B. Most of these programs are based on either solved examples or exercises in the book. Sample output obtained from these programs is also included in files *.out. These output files may be useful to check if the program is running correctly (at least giving the same output!) on your computer. Because of differences in roundoff errors between different compilers and computers the output may not exactly match on two different machines. In most cases the output refers to output on Fortran logical unit 6 which is normally the terminal. This output has been redirected to a file which is included in the directory. In a few cases (anneal.f, bsplin.f, rls.f, smooth.f) the programs directly write to a file on the disk. Of these bsplin.f and smooth.f do not have any significant output on the screen and it is not included in the CD. While for anneal.f and rls.f the screen output is included in addition to the files directly written.

Some Fortran compilers running under Windows operating system require a Fortran file with extension .for rather than .f as required by compilers under Unix. To allow for this, two copies of all Fortran files are provided with names *.f and *.for. Except for the name, these pairs of files are identical. Nevertheless, all links in html files point to files *.f only.

Apart from 117 pairs of Fortran programs and the corresponding output files this directory contains the following files:
 

 kernel   Data file containing kernels for Inverse problem (Example 12.5)
 readme   The readme file
 readme.htm   This file
 xmp.ind   Index of subprograms used in different examples
 xmp.lst   Subject wise list of programs
 xmp.sor   Alphabetic list of programs
 xmplst.htm   Subject wise list of subprograms with HTML links
 xmplsta.htm   Alphabetic list of subprograms with HTML links
